# Đưa code lên GitHub **không cần cài Git**

Bạn có thể dùng trình duyệt để upload code lên repo mà không cần cài Git.

---

## Bước 1: Nén thư mục dự án

1. Mở **File Explorer**, vào thư mục **`d:\demo`**.
2. **Chọn toàn bộ** file và thư mục trong `demo` (Ctrl+A).
   - Không cần chọn thư mục `target` nếu có (có thể bỏ chọn để file zip nhỏ hơn).
3. **Chuột phải** → **Gửi tới** → **Thư mục nén (zip)**.
4. Đặt tên file zip (ví dụ: `Lab-2-OOP.zip`).

---

## Bước 2: Đăng nhập GitHub

1. Vào: **https://github.com**
2. Đăng nhập (hoặc tạo tài khoản nếu chưa có).

---

## Bước 3: Upload vào repo

1. Mở repo: **https://github.com/FIT-dev-AI/Lab-2-OOP**
2. Repo đang trống sẽ có nút **"uploading an existing file"** hoặc **"Add file"** → **"Upload files"**.
3. Kéo thả **toàn bộ thư mục `demo`** (hoặc file zip vừa tạo) vào vùng upload.
   - Nếu bạn upload **file zip**: GitHub sẽ tự giải nén và tạo đúng cấu trúc thư mục.
4. Ở dưới, nhập **Commit message** (ví dụ: `Initial commit: Spring Boot Book API`).
5. Bấm **"Commit changes"**.

Xong. Code của bạn đã nằm trên GitHub.

---

## Lưu ý

- **Không cần cài Git** khi làm theo cách này.
- Sau này khi đã cài Git, bạn có thể clone repo về và dùng `git push` như bình thường.
- Nếu repo thuộc tổ chức **FIT-dev-AI**, bạn cần quyền ghi (write) vào repo đó thì mới upload được.
